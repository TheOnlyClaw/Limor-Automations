# Limor Automations

Monorepo.

- **API:** Fastify + TypeScript (`apps/api`)
- **Web:** React (Vite) + Tailwind CSS (`apps/web`)
- **Specs:** OpenSpec (`openspec/`)

## Dev

Install:

```bash
npm install
```

Run both:

```bash
npm run dev
```

Run API only:

```bash
npm run dev:api
```

Run web only:

```bash
npm run dev:web
```
